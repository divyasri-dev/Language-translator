# Voice Language Translator using Python
A speech-based language translation system built using Python and Google API.
## Features
- Accepts **voice input** from the user
- Converts speech to text
- Supports translation into multiple languages
- User enters a **language code** for the target language
- Speaks the translated output using text-to-speech
## Technologies Used
- Python
- Google Translation API
- Speech Recognition
- Text-to-Speech (TTS)
## How It Works
1. User speaks a sentence.
2. The system listens and converts speech into text.
3. User enters the target language code (for example: `hi` for Hindi, `te` for Telugu, `fr` for French).
4. The text is translated into the selected language.
5. The translated result is spoken aloud by the computer.
## Requirements
Recommended Python version: **Python 3.11**
